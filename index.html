About

Hi, I'm Andrew.

Like many people learning web design, I wasn't born knowing how to write HTML or CSS. I learned by asking questions, making mistakes, rebuilding projects, and slowly discovering how websites really work.

I created this site because I believe it's never too late to learn something new.

Much of the learning material online assumes you already understand the jargon or moves so quickly that beginners can feel left behind. My goal is different. I explain not only how to build websites, but also why things are done a certain way, using clear language and practical examples.

Every project on this site is built step by step. There are no shortcuts, no unnecessary complexity, and no expectation that you already know the answers. We'll build real websites together while learning the skills that professional developers use every day.

Whether you're changing careers, starting a new hobby, returning to learning after many years, or simply curious about how the web works, you're welcome here.

Let's build something you're proud of.
